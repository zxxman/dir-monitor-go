# Dir-Monitor-Go 文档

欢迎来到 Dir-Monitor-Go 项目的文档中心！Dir-Monitor-Go 是一个高性能的文件系统监控工具，可以实时监控指定目录的变化，并在检测到文件事件时触发自定义命令。

> 🏠 **返回项目主页**: [README.md](../README.md) - 查看项目简介、安装和快速开始

本文档中心包含了项目的完整文档，帮助您了解、使用和贡献代码。
