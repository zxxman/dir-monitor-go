# 更新日志

本文档记录了 Dir-Monitor-Go 项目的所有重要变更。

## [3.2.1] - 2025-10-16

### 新增
- 完整的文档体系重构，包括：
  - 全新的 README.md，提供项目概述和快速入门指南
  - 详细的用户指南 (USER_GUIDE.md)，包含安装、配置和使用说明
  - 完整的开发文档 (DEVELOPMENT.md)，包含架构说明和贡献指南
  - API 文档 (API.md)，详细说明所有公共接口
  - 设计文档 (DESIGN.md)，包含系统架构和设计决策
- 文档备份系统，将旧文档迁移至 docs/backup/ 目录
- 配置文件结构优化，支持更多高级配置选项
- 改进的错误处理和日志记录系统

### 改进
- 优化配置验证逻辑，提供更友好的错误信息
- 增强系统稳定性，改进目录监控的可靠性
- 改进并发处理机制，提高多目录监控性能
- 优化内存使用，减少资源占用

### 修复
- 修复配置文件加载时的潜在问题
- 解决特定情况下文件监控丢失的问题
- 修复日志轮转配置不生效的问题

### 安全
- 加强配置文件安全性验证
- 改进命令执行安全性检查
